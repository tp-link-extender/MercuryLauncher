# Mercury Launcher

Launcher for [Mercury Core](https://github.com/tp-link-extender/MercuryCore)-based game creation platforms

## Build

Windows

`dotnet publish --no-self-contained --nologo -c release -o ./out -r win-x64`

Linux

`dotnet publish --no-self-contained --nologo -c release -o ./out -r linux-x64`
